    • 1. 关闭防火墙

    先将linux的防火墙关闭
    1) 重启后生效
    开启： chkconfig iptables on
    关闭： chkconfig iptables off 关闭开机启动

    2) 即时生效，重启后失效
    开启： service iptables start
    关闭： service iptables stop

    3) 查看防火墙状态
    service iptables status

    5.创建hadoop用户

    adduser hadoop
    passwd hadoop

    转到hadoop用户下

    cd /home/hadoop

    6.配置SSH免密码登录

    创建.ssh
    ssh 192.168.187.100
    cd.ssh

    6.1 生成ssh免登录密钥

    创建密钥对
    首先进入.ssh
    ssh-keygen -t rsa
    执行完这个命令后，会生成两个文件id_rsa(私钥)，id_rsa.pub(公钥)
    将公钥拷贝到要免密码登陆的机器上
    cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
    或
    ssh-copy-id master

    7. 配置JDK

    7.1 上传JDK包

    7.2 解压JDK包

    tar -zxvf jdk-7u45-linux-x64.tar.gz -C /解压到的地址

    7.3 配置jdk环境变量

    vi /etc/profile
    在文件末尾添加：
    export JAVA_HOME=/opt/jdk1.7.0_45
    export PATH=$PATH:$JAVA_HOME/bin
    刷新配置：source /etc/profile

    8. 配置hadoop

    8.1上传hadoop包

    8.2 解压hadoop包

    首先在跟目录下创建一个master目录

    mkdir /master

    tar -zxvf hadoop-2.2.0.tar.gz -C /

    8.3 配置hadoop伪分布式（要修改5个文件）

    第一个: hadoop-env.sh
    vim hadoop-env.sh
       export JAVA_HOME=/opt/jdk1.7.0_45

    第二个：core-site.xml
    vim core-site.xml



    fs.defaultFS
    hdfs://master:9000

        <!-- 用来指定hadoop运行时产生文件的存放目录 -->
    <property>    
        <name>hadoop.tmp.dir</name>
        <value>/master/hadoop-2.2.0/tmp</value>
    </property>
<configuration>

    第三个:hdfs-site.xml 

    <configuration>
    <!--用来指定HDFS保存数据副本的数量-->
    <property>
        <name>dfs.replication</name>
        <value>1</value>
    </property>
<configuration>

    第四个：mapred-site.xml(mv mapred-site.xml.template mapred-site.xml) 

    <configuration>  

    <！-- 告诉hadoop以后MR运行在YARN上 -->
    <property>
        <name>mapreduce.framework.name</name>
        <value>yarn</value>
    </property>
</configuration>

    第五个：yarn-site.xml 

    <configuration>  
    <!-- NodeManager获取数据的方式是shuffle -->
    <property>
        <name>yarn.nodemanager.aux-services</name>
        <value>mapreduce_shuffle</value>
    </property>
    <!-- 指定YARN的老大（ResourceManager）的地址 -->
    <property>
        <name>yarn.resourcemanager.hostname</name>
        <value>master</value>
    </property>
</configuration>

    8.4 将Hadoop添加到环境变量中

    export HADOOP_HOME=/master/hadoop-2.2.0
    export JAVA_HOME=/opt/jdk1.7.0_45
    export JRE_HOME=$JAVA_HOME/jre
    export CLASSPATH=$JAVA_HOME/lib:$JRE_HOME/lib:$CLASSPATH
    export PATH=$JAVA_HOME/bin:$JRE_HOME/bin:$PATH:$HADOOP_HOME/bin

    刷新配置
    source /etc/profile

    8.5 初始化HDFS（格式化文件系统）

    hadoop namenode -format(过时了)
    hdfs namenode -format

    8.6 启动HDFS和YARN

    ./start-all.sh(过时了This script is Deprecated. Instead use start-dfs.sh and start-yarn.sh)

    有个小问题（需要多次输入密码）

    通过浏览器的方式验证

    http://192.168.187.185:50070(hdfs管理界面)
    http://192.168.187.185:50030
    http://192.168.187.185:8088(yarn管理界面)

    8.7 测试HDFS

    上传文件到HDFS
    hadoop fs -put /opt/jdk-7u45-linux-x64.tar.gz hdfs://master:9000/jdk

    从HDFS下载文件到本地
    hadoop fs -get hdfs://master:9000/jdk /home/jdk1.7
